## Chart Versioning

- chart version vs app version
- incrementing versions

When to change chart version(Mandatory):

- Template changes (Deployment, Service, Ingress, etc.)
- Default values change
- Chart refactoring or bug fixes
- Helm logic changes

When to change appVersion(Optional):

- New application release
- New container image version
- Bug fix or feature release in the app
